## 🌐 | THE AI LIBRARY PORTAL

A fully rendered **static web portal** generated from every markdown file:

```
ai-library/
├── generate.py         # Reads all markdown, generates the complete site
├── out/                # 🚀 Open me in any browser (975 MB of knowledge)
│   └── index.html      # Start here
└── ...                 # Next.js source files
```

The portal transforms these documents into a browsable, searchable web experience with cross-references, navigation, and a unified reading interface.

> 🔥 **Open `ai-library/out/index.html`** in any browser to experience the Great Library.

## 🚀 | QUICK START

```bash
# 1. Clone the Great Library
git clone https://github.com/hernanda-git/ai-knowledge-library.git

# 2. Browse raw knowledge
cd ai-knowledge-library
ls -la 01-Foundations/   # Pick a wing

# 3. Open the rendered portal
open ai-library/out/index.html   # Or double-click in your file manager

# 4. Regenerate the portal (optional)
cd ai-library && python generate.py   # Rebuilds from all markdown files
```
